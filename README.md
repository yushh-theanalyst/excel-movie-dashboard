# Excel Movie Dashboard

## Project Overview
This project is an Excel-based movie analytics dashboard created using Rotten Tomatoes movie data.  
The dataset contains 16,638 movie records and was analyzed to find useful insights related to movie ratings, genres, runtime categories, and release trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design
- KPI Cards

## Dashboard Features
- Total movie count KPI
- Tomatometer status breakdown
- Movies by rating
- Genre-wise movie analysis
- Runtime category analysis
- Movie release trend over time
- Key business insights section

## Key Insights
- Rotten movies form the largest share of the dataset.
- Drama and Comedy dominate the movie collection.
- Documentaries and Classics receive stronger critic ratings.
- Most movies fall into the medium runtime category.
- Critic and audience preferences show noticeable variation across genres.
